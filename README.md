# Selenium-Automation-Project--Internet-Demo-Site
📄 Overview
This project is an automation testing framework built using Java, Selenium WebDriver, and TestNG, following the Page Object Model (POM) design pattern.
It automates various UI functionalities from The Internet Herokuapp demo website, which contains multiple example pages useful for practicing Selenium automation.

🎯 Features Automated
The following test scenarios are covered:

A/B Testing

Add/Remove Elements

Basic Authentication

Broken Images Detection

Rich Text Editor

Typos Page Verification

Status Codes Validation

Checkbox Handling

Context Menu Actions

Direct Authentication Links

Drag and Drop

Dropdown Selection

Dynamic Controls

Dynamic Loading

File Download

File Upload

Form Authentication

Slider Movement

Mouse Hover Tooltips

Input Field Testing

jQuery UI Elements

JavaScript Alerts Handling

Key Press Event Validation

Multiple Windows Handling

🏗️ Tech Stack
Programming Language: Java

Automation Tool: Selenium WebDriver

Testing Framework: TestNG

Design Pattern: Page Object Model (POM)

Build Tool: Maven

Browser: Chrome (via ChromeDriver)
